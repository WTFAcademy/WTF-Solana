## Part2: Solana Web3.js 极简教程
负责人 0xAA @AmazingAng

Solana web3.js是一个完整而简洁的JavaScript库，可以让开发者便捷地与Solana区块链交互，与以太坊的ethers.js库类似。你需要使用npm安装它：

```bash
npm install @solana/web3.js@1.98.0
```
